<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 1: Verify Memory UI on Profile page</name>

  <what-built>
    Memory section on the Profile page showing per-world memories grouped by world name, a separate "About You" user facts section, individual fact deletion with exit animation, and per-world "Clear all" with inline confirmation.
  </what-built>

  <how-to-verify>
    1. Start dev server: `npm run dev`
    2. Open http://localhost:5173 in browser, log in
    3. Ensure you have at least one world with 5+ messages exchanged (so extraction has fired at least once). If not, chat in a world, navigate away, then return to Profile.
    4. Navigate to Profile page (/profile)
    5. Scroll below the Feedback button — you should see a "Your Memories" section with a Brain icon

    **Check MUI-01 (world memories grouped):**
    - Each world with extracted memories appears as a card with the world name as header
    - Facts are listed with their text, category, and source quote

    **Check MUI-02 (user facts separate):**
    - An "About You" section appears above the world memories
    - It shows global user facts (things the AI learned about you personally)

    **Check MUI-03 (delete individual facts):**
    - Hover over any fact — an X button should appear on the right
    - Click it — the fact should animate out and disappear immediately
    - Refresh the page — the fact should still be gone (persisted to Supabase)

    **Check MUI-04 (clear all per world):**
    - Each world card has a "Clear all" button (red text)
    - Click it — should show "Are you sure? [Yes] [No]" inline
    - Click "Yes" — entire world section disappears
    - Refresh — world no longer appears (row deleted from Supabase)

    **Check MUI-05 (empty states):**
    - If a world has no facts (edge case), it should show "No memories yet"
    - If no memories exist at all, a message like "No memories yet. Chat in a world and characters will start remembering you." should appear
  </how-to-verify>

  <resume-signal>Type "approved" if everything works, or describe any issues you see.</resume-signal>
