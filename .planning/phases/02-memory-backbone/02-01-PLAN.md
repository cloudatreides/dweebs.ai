**EXTRACTION_SYSTEM_PROMPT constant** (exact text):
```
You extract structured memory from group chat transcripts between a user and AI characters.

Given a conversation transcript, extract:
1. worldFacts: up to 5 key facts about what happened in this world (events, topics discussed, running jokes, character dynamics). Each fact has:
   - "fact": 1 sentence describing the fact
   - "source": verbatim quote from the transcript that supports this fact (max 50 chars)
   - "category": one of "event", "preference", "relationship", "topic", "joke"
2. worldSummary: 1-2 sentences describing the overall vibe/history of this world (max 200 chars).
3. userFacts: up to 3 persistent facts about the user that would be relevant in ANY world (name, preferences, interests). Each has:
   - "fact": 1 sentence
   - "source": verbatim quote (max 50 chars)
   - "category": one of "identity", "preference", "interest"

If there are existing facts provided, check for contradictions. If a new fact contradicts an existing one, include ONLY the new fact (it supersedes the old one).

Respond with ONLY valid JSON, no markdown code fences:
{"worldFacts": [...], "worldSummary": "...", "userFacts": [...]}
```

**`extractAndSaveMemory({ chatId, messages, characters, scene })` — exported async function:**
1. Guard: if `!messages || messages.length < 5` return immediately (MEXT-01 threshold guard — the 5-message session count is enforced by the caller in ChatView, but this is a safety net on total messages)
2. Fetch existing facts for contradiction resolution (MEXT-03): `const [existingMemory, existingUserFacts] = await Promise.all([getWorldMemory(chatId), getUserFacts()])`
3. Build transcript from last 30 messages (MEXT-05): `messages.slice(-30).map(m => { if (m.type === 'user') return 'You: ' + m.text; if (m.type === 'character') { const char = characters.find(c => c.id === m.characterId); return char ? char.name.split(' ')[0].toUpperCase() + ': ' + m.text : null } return null }).filter(Boolean).join('\n')`
4. Parse existing facts defensively (handle both string and object): `const existingWorldFactsList = existingMemory?.facts ? (typeof existingMemory.facts === 'string' ? JSON.parse(existingMemory.facts) : existingMemory.facts) : []` and same pattern for existingUserFacts
5. Build userContent string: `"EXISTING WORLD FACTS:\n"` + numbered list of existing world facts (or "None") + `"\n\nEXISTING USER FACTS:\n"` + numbered list of existing user facts (or "None") + `"\n\nCONVERSATION TRANSCRIPT:\n"` + transcript
6. Get auth token: `const { data: { session } } = await supabase.auth.getSession()`, `const token = session?.access_token`. If no token, return silently.
7. Call `/api/chat`: `fetch('/api/chat', { method: 'POST', headers: { 'Content-Type': 'application/json', 'Authorization': 'Bearer ' + token }, body: JSON.stringify({ model: 'claude-haiku-4-5-20251001', max_tokens: 500, system: EXTRACTION_SYSTEM_PROMPT, messages: [{ role: 'user', content: userContent }] }) })`
8. If `!response.ok`, `console.warn('Memory extraction API failed:', response.status)` and return
9. Parse response: `const data = await response.json()`, `const rawText = data.content?.[0]?.text || ''`
10. Parse JSON via `parseExtractionJSON(rawText)`. If null, `console.warn('Memory extraction JSON parse failed:', rawText.slice(0, 200))` and return
11. Destructure: `const { worldFacts, worldSummary, userFacts } = parsed`
12. Write world memory: if `worldFacts?.length > 0 || worldSummary`, call `await upsertWorldMemory(chatId, (worldFacts || []).slice(0, 10), worldSummary || '')`
13. Write user facts: if `userFacts?.length > 0`, call `await upsertUserFacts((userFacts || []).slice(0, 10))`
14. Wrap entire function body in try/catch. Catch logs `console.warn('Memory extraction failed (non-blocking):', err.message)` and returns — never throws.

**`parseExtractionJSON(raw)` — internal (not exported) function:**
1. Try `JSON.parse(raw)` — return if succeeds
2. Strip markdown fences: `raw.replace(/```json?\s*/g, '').replace(/```\s*/g, '').trim()` — try JSON.parse again
3. Regex extract `{...}`: `stripped.match(/\{[\s\S]*\}/)` — try JSON.parse on match[0]
4. Return null if all fail

**`buildMemoryBlock(worldMemory, userFacts, maxChars = 500)` — exported function:**
1. If `!worldMemory && !userFacts` return `''`
2. Build block string starting empty
3. If worldMemory exists: parse facts defensively (`typeof worldMemory.facts === 'string' ? JSON.parse(worldMemory.facts) : (worldMemory.facts || [])`), get summary (`worldMemory.summary || ''`). If summary or facts exist, append `'MEMORY OF THIS WORLD:\n'` + summary + `'\n'` (if summary) + `'Key facts: '` + facts mapped to `.fact` joined by `'. '` + `'.\n'` (if facts exist)
4. If userFacts exists: parse facts same pattern. If facts exist, append `'\nABOUT THE USER:\n'` + facts mapped to `.fact` joined by `'. '` + `'.\n'`
5. Truncation if block.length > maxChars: first try removing the ABOUT THE USER section via `block.replace(/\nABOUT THE USER:\n[\s\S]*$/, '')`. If still over, hard truncate to `maxChars - 3` and append `'...'`
6. Return block
